# 🌐 ConnectSphere

ConnectSphere is a full-stack social media web application built using the MERN stack. It allows users to connect, share posts, like, comment, and interact in real-time with a modern and responsive interface.

---

## 🚀 Features

- 🔐 User Authentication (JWT-based)
- 📝 Create, edit, and delete posts
- ❤️ Like and 💬 comment on posts
- 👥 Follow / Unfollow users
- 🟢 Real-time online users (Socket.IO)
- 🔔 Notifications system
- 📱 Fully responsive UI
- ⚡ Efficient state management using Redux Toolkit

---

## 🛠️ Tech Stack

**Frontend:**
- React.js
- Redux Toolkit
- Tailwind CSS

**Backend:**
- Node.js
- Express.js

**Database:**
- MongoDB Atlas

**Other Tools:**
- Socket.IO (real-time communication)
- JWT (authentication)
- Cookie-parser
